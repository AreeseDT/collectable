selectView

  • selectView<T>(state: ListState<T>, ordinal: number, asWriteTarget: boolean): View<T>
    • Defined in list/traversal.ts:61

    Selects and returns either the left or the right view for further operations at the specified ordinal position. The view is selected with a preference for preserving the position of the last view that was written to, so that the reading and writing of views will implicitly optimise itself according to the way the list is being used.
